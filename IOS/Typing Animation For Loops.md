# For loops

```swift
let titleText = "👨‍💻Testletter"

for letter in titleText {
  print(letter)
}
```

```swift
let fruitBasket = ["Apple", "Peer"]

for fruit in fruitBasket {
  print(fruit)
}
```

fruitBasket라는 array의 모든 요소들이 print된다.
